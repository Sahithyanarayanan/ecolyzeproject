Dataset Information

This section provides details about the gene sequence dataset used for E.coli disease prediction. The dataset consists of genomic sequences processed using trimer (3-mer) analysis and machine learning techniques to identify patterns and predict disease outcomes.

Dataset Overview

This dataset contains E.coli gene sequences used for disease prediction. Each sequence is analyzed using trimer (3-mer) patterns to identify biological signals.

Data Collection

Gene sequences are collected from biological sources such as NCBI databases. Files are stored in .fna format for processing.

Feature Extraction

Trimer frequency features are extracted from gene sequences to convert biological data into numerical format for machine learning models.

Data Preprocessing

Data is cleaned and normalized using scaling techniques to improve model accuracy and ensure consistency.

Train-Test Split

The dataset is divided into training and testing sets to evaluate model performance and prediction accuracy.
